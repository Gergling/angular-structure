# angular-structure
Documentation and examples for structuring angularjs in a way that is scalable and maintainable.

## File Structure

File structure format is as follows:

* [module-name]/
  * module.js
  * [component-type]/
    * [component-name].js
  * partial/
    * [partial-name].html
    * directive/
      * [directive-name].html
    * view/
      * [view-name].html
